extract_simulation_mesh(clean_up: bool = False) → None#

Update the model.

Examples

Processes a model from the public database and generates a simulation mesh

  1. Instantiate object that stores relevant information for the preprocessor

>>> info = ModelInfo(
               database="Strocchi2020",
               path_to_case = "01.case",
               work_directory = "workdir",
               mesh_size = 1.5
               )

  1. Instantiate the model object

>>> model = FullHeart(info)

  1. Run the extract simulation mesh method

>>> model.extract_simulation_mesh()

  1. Save model to disk

>>> model.dump_model()

  1. Print info about the model

>>> model.print_info()
