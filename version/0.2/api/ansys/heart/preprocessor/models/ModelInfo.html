ModelInfo#

class ModelInfo(input: pathlib.Path | str | pyvista.PolyData | pyvista.UnstructuredGrid = None, scalar: str = 'part-id', part_definitions: dict = None, work_directory: pathlib.Path = '.', path_to_simulation_mesh: pathlib.Path = None, mesh_size: float = 1.5, add_blood_pool: bool = False)#

Contains model information.

Overview#

clean_workdir

Remove files with extension present in the working directory.

create_workdir

Create the working directory if it doesn’t exist.

dump_info

Dump model information to file.

input

Input to the workflow.

scalar

Scalar field name with part/surface ids.

part_definitions

Part definitions.

workdir

Path to the working directory.

path_to_simulation_mesh

Path to simulation(in .vtk format).

path_to_model

Path to model (in .pickle format).

mesh_size

Mesh size used for remeshing.

add_blood_pool

Flag indicating whether to add blood to the cavities.

model_type

Deprecated dummy value of model type.
